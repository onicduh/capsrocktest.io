Extract the downloaded source code zip file.
Copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory.
Browse the PHPMyAdmin in a browser. i.e. http://localhost/phpmyadmin
Create a new database named fos_db.
Import the provided SQL file. The file is known as fos_db.sql located inside the database folder.
Browse the Online Food Ordering System V2 in a browser. i.e. http://localhost/fos/ and http://localhost/fos/admin for the admin side.
